Starfruit Translator Bundle

# Installation

```bash
    composer require starfruit/translator-bundle
```

- Update `config/bundles.php` file:

```bash
    return [
        ....
        Starfruit\TranslatorBundle\StarfruitTranslatorBundle::class => ['all' => true],
    ];
```
